# Java Practice Projects

A collection of Java practice programs demonstrating core Object-Oriented Programming (OOP), file handling, exception handling, collections, interfaces, abstract classes, inheritance, polymorphism, and encapsulation concepts. The repository contains multiple standalone Java applications created for learning and academic practice. 

## Projects Included

* Smart Warehouse
* Student Grade Filter
* Invalid Age Protector
* Daily Sales Log
* Device Manager
* Duplicate Finder
* Safe Math Calculator
* Product Inventory
* Log File Reader
* ATM Simulation
* Vehicle Fleet
* Survey Analyzer
* Library System
* Config Loader
* Employee Payroll

## Topics Covered

* Object-Oriented Programming (OOP)

## How to Run

1. Clone the repository:

```bash
git clone https://github.com/your-username/java-practice-projects.git
```

2. Open the project in IntelliJ IDEA or any Java IDE.

3. Compile and run any `.java` file individually:

```bash
javac FileName.java
java FileName
```

## Learning Objectives

This repository was created to strengthen understanding of Java programming fundamentals through practical examples and mini-projects. The programs focus on applying theoretical concepts to real-world scenarios such as banking systems, inventory management, payroll processing, vehicle rentals, and file management.
